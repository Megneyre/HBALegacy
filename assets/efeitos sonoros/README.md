# Efeitos sonoros

O sistema procura estes arquivos:

```text
selecionar-jogador.mp3
encaixar-jogador.mp3
spin.mp3
vitoria.mp3
derrota.mp3
```

Caminho-base usado pelo código:

```text
assets/efeitos sonoros/
```

Arquivos ausentes não interrompem o jogo. Use efeitos próprios ou licenciados para publicação web.

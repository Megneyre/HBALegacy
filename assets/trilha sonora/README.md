# Trilha sonora

Coloque nesta pasta a música principal do jogo com este nome exato:

```text
trilha-principal.mp3
```

Caminho usado pelo código:

```text
assets/trilha sonora/trilha-principal.mp3
```

O arquivo não acompanha o projeto para evitar problemas de direitos autorais. Use uma faixa própria ou licenciada para jogos e publicação web.
